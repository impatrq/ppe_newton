# Dailies del Proyecto

Esta carpeta contiene los informes diarios (**dailies**) de las reuniones del equipo, donde se documenta
El progreso, las actividades realizadas y los objetivos de cada día durante el desarrollo del proyecto.

## Contenido

Cada archivo sigue la convención:

daily-XX.md

- `XX` → número consecutivo de la reunión diaria.
- Incluye: miembros presentes, trabajo realizado, expectativas del día y observaciones.
Se mantiene la plantilla original planteada

## Cómo usarlo

- Abrir cualquier `daily-XX.md` para ver los detalles de la jornada correspondiente.
- Los informes permiten llevar un registro histórico del proyecto y facilitar la coordinación del equipo.
- Útil para futuros miembros o para presentar avances en informes de prácticas profesionalizantes.
